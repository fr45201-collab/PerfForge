# PerfForge

Measure. Analyze. Optimize.

PerfForge is an advanced performance profiling and optimization platform built with:

- C (Profiling Engine)
- Python (Analytics & Visualization)
- PostgreSQL (Storage)
- React (Dashboard)
- Docker (Containerization)
- Groq AI (Optimization Insights)

## Features

- CPU Profiling
- Memory Tracking
- Thread Monitoring
- Hotspot Detection
- Flame Graphs
- Performance Reports
- AI-Powered Optimization Suggestions

## Architecture

C Profiler Agent
↓
JSON Reports
↓
Python Analytics
↓
PostgreSQL
↓
React Dashboard
